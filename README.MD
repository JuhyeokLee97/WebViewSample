## ⚒️사용 기술
- `Android`, `Kotlin`
- `View Binding`, `SwipeRefreshLayout`
## 🔎기능
- 웹 **뒤로 가기**, **앞으로 가기** 기능 구현
- **홈 버튼** 기능 구현
- 웹 페이지 이동 시, **로딩 정도**(progress bar) 표기 구현
